Software Defined Media

Object-based digital media

Media data is decoupled from audio-visual data and 3D meta-data, which represents multiple objects that exist in 3D space, and transmitted to remote locations for the reproduction of 3D space.

Software defined media space

The software manages the audio-visual objects in the virtual 3D space and renders images and sounds suitable for the rendering environments.

Internet by design

Targeting new research areas and markets involving Internet-by-design audio-visual environments
